# 🛒 Amazon Clone (Frontend Project)

## 📌 Project Overview

This is a simple Amazon Clone frontend project built using only **HTML and CSS**. It replicates the basic UI of the Amazon website including homepage layout, navigation bar, and product sections.

---

## 🚀 Features

* Amazon-like homepage design
* Navigation bar layout
* Product cards section
* Responsive UI design (basic)
* Static frontend design using HTML & CSS

---

## 🛠️ Technologies Used

* HTML
* CSS

---

## 📁 Project Structure

Amazon_clone/
│── index.html
│── style.css
│── images/

---

## 🎯 Objective

The main goal of this project is to practice frontend development skills and understand how basic e-commerce website layouts are created using HTML and CSS.

---

## 📚 Learning Outcomes

* Improved HTML structure building
* CSS styling and layout design
* Understanding webpage UI structure
* Basic e-commerce website design practice
